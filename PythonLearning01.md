### 4.字符串的格式化
```python
name = 'Python'
age = 21
name1 = 'Python1'
age1 = '22'

new_str = "我是" + name + "," + "今年" + str(age) + "岁了"
new_str_1 = "我是%s,今年%d岁了" % (name, age) # python2
new_str_2 = "我是{},今年{}岁了".format(name, age)
new_str_3 = "我是{name},今年{age}岁了".format(name = 'Python2', age = '23')
new_str_4 = "我是{name},今年{age}岁了".format(name = 'Python2', age = 23)
new_str_5 = f"我是{name1},今年{age1}岁了" # python3.6 后面三种比较常用

print(new_str)
print(new_str_1)
print(new_str_2)
print(new_str_3)
print(new_str_4)
print(new_str_5)
```

### 5.布尔型
```python
# python shell
# bool类型的值只有True和False两种，注意True和False的开头字母必须大写
>>> a = True 
>>> a
True
>>> type(a)
<class 'bool'>
>>> a = '' # 定义空字符串
>>> a
''
>>> print(a)

>>> a = None # 定义空，注意与上种比较
>>> a
>>> print(a)
None
>>> a is None # is None为判断语句，返回值为bool类型
True
>>> b = a is None
>>> b
True
>>> type(b)
<class 'bool'>
```
